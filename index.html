Template Boilerplate - Open

A quickstart you can use for new projects, to help setup a simple folder structure and some other useful defaults.

To begin clone down the project like so, replacing "my-project-name" with your project name.

git clone -o template-boilerplate https://github.com/ReallyEasyCart/template-boilerplate.git my-project-name

Next you'll want to setup your own private repository for your template work, either on Bitbucket (Free) or here on Github (Monthly Cost)

Once you've created your project it should give you the commands to use for and exisiting project to add the repository, then run: git push -u origin master to push back up to your own project.
